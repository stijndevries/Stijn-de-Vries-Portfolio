About me and this website.

First off, thanks for taking a look at my website. I put a lot of time into creating it.

My previous website was made in Sparkle, a fantastic WYSIWYG tool for designing and developing websites. This time around I was interested in making it myself from the ground up.

I got a little obsessed about speed and learned a lot about the nitty-gritty of web development while making this website.

How it's made.

I engineered this website to be fast. No trackers, no JavaScript, and no bitmap images. It's just HTML, Sass, and SVGs.

Responsiveness is added using CSS media queries and Flexbox.

The typeface this is used on this website is Eudoxus Sans, an Open Source typeface that I created based on +Jakarta Sans.
